MGAP Search API
/api/search

Примечание:
Запрос: https://docs.pipl.com/reference#person
Ответ: https://docs.pipl.com/reference#response

Пример:
curl --location --request POST 'http://search.mgap.ru:8081/api/search' \ --header 'Content-Type: application/json' \ --data-raw '{"names":[{"first":"Kirill","last":"Lazarenko"}],"addresses":[{"raw":"Moscow, Russia"}],"phones":[],"emails":[{"address":"clark.kent@example.com"}]}'
